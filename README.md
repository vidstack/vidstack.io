# vidstack.io

Vidstack uses [SvelteKit](https://kit.svelte.dev) for its site.

1. Clone this repo

```bash
git clone git@github.com:vidstack/vidstack.io.git --depth=1
```

2. Go into the project's root directory

```bash
cd vidstack.io
```

3. Install JS dependencies

```bash
npm i
```

4. Start the dev server

```bash
npm run dev
```
